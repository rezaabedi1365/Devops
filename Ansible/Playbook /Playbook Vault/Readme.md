##### Playbook Vault
Create an encrypted file
```
ansible-vault create apache-playbook.yaml
```
Decrypt a file
```
ansible-vault decrypt apache-playbook.yaml
```
Encrypt a file
```
ansible-vault encrypt apache-playbook.yaml
```
